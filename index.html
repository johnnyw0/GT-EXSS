O XSS

Estima-se que mais de 60% dos sítios Web têm algum tipo de vulnerabilidade XSS

No XSS, um atacante explora vulnerabilidades de sítios Web legítimos para executar trechos de código maliciosos nos navegadores dos usuários legítimos. As vulnerabilidades exploradas pelos atacantes são em sua grande maioria encontradas em campos de sítios Web que permitem a entrada de dados pelos usuários legítimos e, em seguida, retornam alguma informação sobre os dados de entrada para esses usuários.

O Emulador Educativo

Interface do
usuário

Uma interface intuitiva que possa atender a usuários de diferentes níveis de habilidade, oferecendo orientações claras e recursos explicativos.

Catálogo de atividades

Cada atividade é composta por diferentes tarefas: uma introdução teórica sobre o assunto da atividade, seguida de procedimentos práticos para realização de testes de vulnerabilidade XSS.

Análise de vulnerabilidades

O núcleo do emulador. É o módulo onde estão hospedadas as páginas com vulnerabilidades que serão acessadas nas atividades práticas.

Relatório
técnico

Após interagir com o laboratório, o aluno receberá um feedback sobre a atividade proposta e serão recompensados com medalhas de conclusão

Sobre as atividades

Progressão controlada

Cada tarefa será desbloqueada progressivamente, condicionada à conclusão do anterior. A liberação gradual das atividades permite que os usuários se familiarizem de forma intuitiva com a resolução das emulações de XSS. Esse método visa garantir que o conteúdo seja absorvido de maneira eficiente, impedindo que o usuário avance para as seções subsequentes sem possuir o conhecimento prévio necessário.

Diferentes níveis

Cada atividade é associada a um nível de conhecimento necessário para o usuário realizá-la. Num primeiro momento, serão desenvolvidas atividades para três níveis: básico, intermediário e avançado. Cada um desses níveis estará contido em uma trilha de aprendizado descrita no menu inicial da plataforma, através da qual será possível acessar as respectivas atividades e tutoriais. O conteúdo será liberado ao usuário sequencialmente conforme ele completa as etapas das atividades propostas, sendo assim, para acessar o nível 6 intermediário será necessário que o aluno complete as atividades propostas no nível básico, da mesma forma para acesso ao nível avançado.
